# E2Q 报价功能集合


## 描述

接口的数据，都是通过 Kafka 发送给 E2Q 系统使用.<br />
接口协议请看: [e2q_protoco](https://github.com/E2Quant/e2q_doc/blob/main/docs/e2q_protocol.md)

## 介绍

### 目前拥有的集合类型有

* ITCH: Nasdaq Stock Market  data messages.
* TickForPy: Python for hdf5 Stock Market data
* xtp_zts: 中泰证券 xtp 接口
* EAQS: 东亚前海证券-极速交易 接口
